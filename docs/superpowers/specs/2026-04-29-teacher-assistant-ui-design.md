# 教师助手 UI 设计规范

## 概述

基于参考截图风格，对教师助手应用进行 UI 重构，实现统一、现代的视觉设计系统。

## 设计原则

- **简洁清晰**：信息层次分明，重点突出
- **统一一致**：颜色、间距、圆角保持统一
- **友好温和**：紫色主色调，圆角设计，视觉柔和

## 布局规范

### 侧边栏
- 宽度：240px
- 背景：#f8f9fa
- 选中态：左侧 3px 紫色边框 + 淡紫背景
- 菜单项：高度 48px，圆角 8px，边距 8px 16px
- Logo 区：高度 80px，底部边框分隔

### 顶部栏
- 高度：64px
- 背景：白色
- 底部边框：1px solid #e5e7eb
- 搜索框：圆角 24px，背景 #f3f4f6，高度 40px
- 用户信息：头像 + 姓名 + 角色

### 主内容区
- 边距：32px
- 页面标题：24px 加粗 + 副标题 14px 灰色
- 操作按钮区：右上角

## 组件规范

### 按钮
**主按钮**
- 背景：#6366f1
- 文字：白色，14px
- 高度：36px
- 内边距：0 16px
- 圆角：8px
- Hover：#818cf8

**次按钮**
- 背景：白色
- 边框：1px solid #e5e7eb
- 文字：#374151
- Hover：背景 #f9fafb

**图标按钮**
- 尺寸：32px 圆形
- Hover：背景 rgba(99,102,241,0.08)

### 卡片
- 背景：白色
- 圆角：12px
- 阴影：0 1px 3px rgba(0,0,0,0.05)
- 内边距：24px
- 边框：1px solid #e5e7eb

### 表格
- 外层包裹：卡片容器
- 表头：背景 #f9fafb，文字 #6b7280，12px
- 行高：56px
- 行悬停：背景 #f9fafb
- 斑马纹：可选

### 标签
- 圆角：6px
- 内边距：4px 8px
- 字号：12px
- 学科标签：带背景色

### 弹窗
- 圆角：16px
- 背景：白色
- 遮罩：rgba(0,0,0,0.5)
- 标题：18px 加粗
- 内容区：padding 24px
- 底部按钮区：padding 16px 24px

## 页面设计详情

### 仪表盘
- 欢迎卡片：紫色渐变背景
- 统计卡片：4 列网格，图标 + 数字 + 标签
- 快捷入口：图标网格
- 最近动态：时间线列表

### 学生管理
- 页面标题 + 副标题
- 搜索框（圆角长条）+ 统计
- 表格：头像(姓) + 姓名 + 年级 + 性别(标签) + 操作
- 性别：男蓝色 / 女粉色
- 添加弹窗：表单字段（姓名、学号、班级、年级、性别）

### 表现记录
- 搜索过滤栏
- 右侧快捷标签面板
- 记录卡片列表：头像、姓名、时间、内容、标签
- 统计卡片：本月表扬、待关注、记录数
- 添加按钮：右上角紫色

### 评语生成
- 统计栏：总学生数、已生成、待生成
- 操作按钮：生成设置、AI 生成评语（紫色）
- 表格：学生、近期成绩（学科标签）、表现记录数、评语预览、状态、操作
- 状态标签：已生成（绿）、待生成（橙）
- 导出按钮：Word、Excel

## 动画与交互

- 过渡时间：200ms
- 缓动函数：ease-out
- Hover：透明度或背景色变化
- 页面切换：淡入淡出

## 响应式断点

- Desktop：> 1280px（默认）
- Tablet：768px - 1280px
- Mobile：< 768px（侧边栏收起）
